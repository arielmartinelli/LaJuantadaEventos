# La Juntada Eventos - Landing Page

Landing page moderna, dinámica e interactiva para **La Juntada**, un servicio gastronómico y de catering premium especializado en casamientos, cumpleaños y eventos corporativos o sociales.

El sitio está desarrollado con tecnologías web estándar y cuenta con una temática visual centrada en el color naranja característico de la marca, además de una estructura totalmente adaptable (mobile-first).

## 🚀 Características

- **Diseño Moderno & Temática Naranja**: Interfaz limpia con toques de glassmorphism y detalles pulidos en base a la identidad de marca.
- **Logotipo Vectorial SVG**: Integración en línea que se escala en alta definición y cambia de color dinámicamente.
- **Cotizador de Presupuestos Online**: Calculadora interactiva que calcula presupuestos en tiempo real basados en la cantidad de invitados, tipo de evento y servicios opcionales.
- **Integración Directa con WhatsApp**: Enlace dinámico que envía la cotización en tiempo real formateada como mensaje de WhatsApp.
- **Carrusel de Fotos**: Slider de platos gastronómicos autoejecutable y responsivo con soporte táctil (swipe).
- **Galería de Fotos Filtrable con Lightbox**: Portafolio dinámico por categorías (Casamientos, Cumpleaños, Corporativos) con visor modal a pantalla completa.
- **Formulario de Contacto**: Formulario adaptativo con validaciones de campos y enlace de derivación directa.
- **Optimización Mobile-First**: Grilla adaptada a celulares mostrando dos tarjetas por fila para mejorar la fluidez de lectura.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura de marcado semántica y SEO optimizada.
- **Vanilla CSS3**: Estilos customizados con variables CSS, transiciones fluidas y grillas responsivas.
- **JavaScript (ES6)**: Lógica nativa de interacción sin librerías pesadas externas.
- **Google Fonts**: Fuentes `Outfit`, `Pacifico` y `Caveat` para las tipografías del sitio.
- **FontAwesome Icons**: Íconos interactivos e ilustrativos.

## 💻 Visualización Local

Para ver el proyecto de manera local, simplemente clona este repositorio y abre el archivo `index.html` en cualquier navegador:

```bash
# Clonar repositorio
git clone https://github.com/arielmartinelli/LaJuantadaEventos.git

# Entrar a la carpeta
cd LaJuantadaEventos

# Abrir index.html (o doble clic en el explorador)
# O iniciar servidor local rápido con Python:
python -m http.server 8000
```
